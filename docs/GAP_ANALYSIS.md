## 風險提醒

| 風險 | 影響 | 因應 |
|------|------|------|
| 14 個佔位 API 被誤認為完成 | 高 | 每個回傳 `message:'待實作'` |
| 21 個 Mock 頁面使用者以為是真的 | 高 | 需加「Demo 模式」提示 |
| 沒有種子資料，登入後看到空白 | 中 | 優先建 seed |
| S3 未設定，上傳功能不能用 | 中 | 先用本地存檔 fallback |
| LINE API 未設定，推播不能用 | 低 | 先用 console.log |

## 一句話總結

**地基打好了（DB 38表 + 權限 + Docker），主結構蓋好了（31 頁面 + 19 路由），但只有 5 間房間真正通水通電（auth/users/leads/cases/finance）。要正式入住還需要 6-8 週把剩下的房間裝修好。**
